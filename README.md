# Code Alpha Bioinformatics Internship – Task 1

## Project Title
Comparative Protein Sequence Analysis of PIGBOS1 using BLASTp

## Objective
To analyze the PIGBOS1 protein sequence from Homo sapiens and compare it across multiple organisms to understand evolutionary conservation and functional importance.

## Methodology
- Protein sequence retrieved from UniProt (Accession: A0A0B4J2F0)
- BLASTp analysis performed using NCBI BLAST
- Non-redundant protein database (nr) used for comparison
- Evaluation based on:
  - Percentage Identity
  - Query Coverage
  - E-value
  - Bit Score

## Key Findings
- PIGBOS1 is highly conserved across mammalian species
- 100% query coverage observed across all alignments
- No gaps detected in alignments
- Strong conservation suggests functional importance

## Conclusion
The protein PIGBOS1 shows strong evolutionary conservation, indicating a likely essential biological role.
